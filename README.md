# asteroids
Asteroids remake

Built with Unity 6000.0.63f1

Instructions:
- Open Scenes/Main to play the game
- Tweak configurations found in Configs/ to change gameplay
- Controls: 
    + Thrust: W or Up arrow
    + Turn right: D or Right arrow 
    + Turn left: A or Left arrow
    + Fire: spacebar
- When in play mode and "Game Over" is displayed, press any of the inputs keys to start a new game
